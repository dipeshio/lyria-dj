# lyria-music-controller

Modern music controller web application built with Vite, React, and Tailwind CSS. Features a clean interface for music playback control.

## Tech Stack
- Vite (build tool)
- React
- Tailwind CSS
- PostCSS

## Installation
```bash
npm install
```

## Development
```bash
npm run dev
```

## Build
```bash
npm run build
```
